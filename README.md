bend2mod
---

## Description

A [midiglue](https://sigboost.audio/midiglue/) patch that converts MIDI pitch bend messages to MIDI modulation messages

<img src="bend2mod.png" width="500px">

## Control

* Button A: Pitch Bend mode
* Button B: Modulation Wheel mode
